# Workshop Registration & Confirmation App

A React-based workshop registration system built for the 
Naan Mudhalvan Arts Internship 2026 at Adroit Technologies.

## Features
- Registration form with input validation
- Duplicate email prevention
- Confirmation modal on successful registration
- Persistent participant list using localStorage

## Tech Stack
React 19, Vite, JavaScript, CSS3

## Run Locally
npm install
npm run dev
